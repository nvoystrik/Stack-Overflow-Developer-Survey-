# Статистичний аналіз результатів опитування розробників Stack Overflow за допомогою Pandas

##  Огляд проєкту
Цей проєкт присвячений дослідженню та аналізу великого датасету глобального опитування розробників від **Stack Overflow** за допомогою мови програмування Python та бібліотеки Pandas. 

Мета роботи — провести розрахунок мір центральної тенденції, оцінити повноту даних, виявити ключові метрики популярності мов програмування (на прикладі Python), проаналізувати поширеність віддаленої роботи, освітні тренди серед програмістів, а також здійснити географічний аналіз річної фінансової компенсації спеціалістів.

##  Реалізований аналітичний функціонал та логіка коду

В межах аналізу великого масиву даних (`survey_results_public.csv`) та його схеми опису (`survey_results_schema.csv`) було виконано такі аналітичні завдання:

1. **Дослідження обсягу вибірки:** Очищено датасет від дублікатів та зафіксовано фінальну кількість унікальних респондентів — **49,191 особа**.
2. **Аналіз повноти відповідей:** За допомогою логіки множин (`set.intersection`) автоматично визначено перелік спільних запитань між схемою та анкетами. Проведено перевірку та фільтрацію рядків без пропусків (`.dropna`), що дозволило оцінити чистоту отриманих анкет.
3. **Розрахунок мір центральної тенденції:** Обчислено статистичні показники досвіду роботи (`WorkExp`) респондентів:
   * **Середнє значення (Mean):** 13.37 років.
   * **Медіана (Median):** 10.00 років.
   * **Мода (Mode):** 10.00 років.
4. **Оцінка поширеності віддаленої роботи:** Застосовано фільтрацію за критерієм типу зайнятості (`RemoteWork == 'Remote'`), що дозволило виділити сегмент із **10,931 розробника**, які працюють повністю віддалено.
5. **Аналіз освітніх трендів:** Досліджено шляхи навчання програмуванню (`LearnCode`). Виявлено, що абсолютна кількість респондентів, які обрали онлайн-курси як основне джерело навчання, становить **10,973 особи**, що дорівнює **32.70%** від усієї кількості тих, хто відповів на це запитання.
